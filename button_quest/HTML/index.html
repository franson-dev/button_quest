Bem-Vindo ao Button Quest!

Nosso foco é dar uma explicação incial e básica sobre algumas Tags do HTML.
HTML (HyperText Markup Language / Linguagem de Marcação de Hipertexto) é um bloco de construção padrão de páginas da Web, é ele quem vai organizar e estruturar um site. Para isso, o HTML utiliza diversas ferramentas e elementos, aqui veremos 4 coisas que podemos fazer com essas ferramentas, são elas: Formulário, Vídeos na página, Listas e Tabelas.

Para que possamos criar todos esses elementos, devemos entender o que de fato são esses elementos. Normalmente, em HTML, um elemento é, no geral, tudo que está dentro de uma Tag (salvo casos como o elemento <br>, que é usado para pular uma linha).
Elementos podem conter outros elementos, tanto que todos os elementos de um site estão contidos no elemento "HTML".











Uma curiosidade sobre o html é que muitos acham que ele é um tipo de linguagem de programação, mas ele não é! Como foi citado no início desta página, o HTML é uma linguagem de marcação de hipertexto.
Além de você não precisar de conhecimento formal para aprender e trabalhar na área, a programação é um conhecimento como qualquer outro. Você precisará de muito estudo e dedicação, mas é possível que qualquer pessoa aprenda.

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
Neste exemplo, vemos 4 elementos de HTML: html, body, h1 e p.
É a partir deste conceito que vamos construir nossos exemplos ao longo das explicações.
Veremos que, basicamente, todas as ferramentas aqui utilizadas serão elementos que contém outros elementos.
