Listas

Listas são muito úteis em diversos tipos de contexto, seja para organizar informações do dia-a-dia, fazer alguma checklist, sumários, organizar tarefas ou para qualquer uso pessoal e/ou profissional. Em Html, listas podem ser feitas para todos esses fins, vejamos alguns exemplos:

Ordenadas

Para começar, veremos a Lista Ordenada que corresponde a tag <ol> (Ordered List).

Exemplo no HTML

<ol>
   <li> Item 1 </li>
   <li> Item 2 </li>
   <li> Item 3 </li>
</ol>
Resultado

  1. Item 1
  2. Item 2
  3. Item 3

Como podemos ver, a nossa lista fica dentro da tag <ol>, que indica onde começa e onde termina a lista ordenada.
Dentro dela, vemos que há uma nova tag, a tag <li> (List Item), esta tag serve para identificar cada item dentro da lista.


      Podemos também colocar dois atributos dentro da tag <ol>, são eles: type e start.

  • type
    •   Indica o tipo do marcador que vai ser utilizado na lista.
      Há cinco tipo de marcadores: Numeral (padrão) [1], Letras (Maiúsculas) [A], Letras (Minúsculas) [a], Números (Romano Maiúsculos) [I] e Número (Romano Minúsculos) [i].

  • start
    •   Indica de onde uma lista ordenada irá iniciar, por padrão sempre começa no número 1.
      Veja alguns exemplos:
Exemplo com Type

<ol type = "i">
   <li> Item 1 </li>
   <li> Item 2 </li>
   <li> Item 3 </li>
</ol>
<ol type = "A">
   <li> Item 1 </li>
   <li> Item 2 </li>
   <li> Item 3 </li>
</ol>
Resultado

  1. Item 1
  2. Item 2
  3. Item 3

  1. Item 1
  2. Item 2
  3. Item 3
Exemplo com Start

<ol start = "50">
   <li> Item 1 </li>
   <li> Item 2 </li>
   <li> Item 3 </li>
</ol >

<ol start = "50" type = "I">
   <li> Item 1 </li>
   <li> Item 2 </li>
   <li> Item 3 </li>
</ol >
Resultado

  1. Item 1
  2. Item 2
  3. Item 3

  1. Item 1
  2. Item 2
  3. Item 3


Desordenadas

Além das listas ordenadas, temos também a lista desordenada, cuja qual não numera seus itens e sua tag é <ul> (Unordered List). Normalmente vemos este tipo de lista em slides de aprensentações, onde, geralmente, fica uma bolinha antes do texto, vamos a um exemplo:

Exemplo no HTML
<ul>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ul>
Resultado
  • Item 1
  • Item 2
  • Item 3

Listas desordenadas não precisam de um inicio (start).
Assim como nas ordenadas, você pode, com o CSS, alterar o tipo de marcador da lista desordenada.
Os tipos de marcadores são: Disco (padrão) [disc], Círculo [circle], Quadrado [square] e Vazio (sem marcador) [none].
Veja alguns exemplos:

Exemplo no HTML
<ul style = "list-style-type: circle;">
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ul>

<ul style = "list-style-type: circle;">
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ul>
Resultado
  • Item 1
  • Item 2
  • Item 3

Resultado
  • Item 1
  • Item 2
  • Item 3
