Formulários

A flexibilidade dos formulários HTML fazem com que sejam uma das estruturas mais complexas do HTML, você pode construir qualquer tipo básico de formulário usando elementos e atributos exclusivos de formulários. Assim, usar a estrutura correta ao criar um formulário HTML o ajudará a garantir que o formulário seja utilizável e acessível.



Codigo:

<form action = "" method = "POST" name = "formPessoa">
<b>Bem-vindo ao meu formulário. Preencha os dados abaixo e depois envie-os para mim.
Muito obrigado.</b>


Nome completo: <input type = "text" name = "Nome" size = "35">

Seu e-mail: <input type = "text" name = "Email" size = "30">

Sexo: <input type = "RADIO" name = "Sexo" value = "F"> Feminino
<input type = "RADIO" name = "Sexo" value = "M"> Masculino


Bens: <input type = "checkbox" name = "bens" value = "C">Casa
<input type = "checkbox" name = "bens" value = "A">Automóvel
<input type = "checkbox" name = "bens" value = "M">Micro


Qual o seu hobby preferido? <select name = "Hobby">
<option value = "Livros"> Ler livros
<option value = "Música"> Ouvir música
<option value = "Cinema"> Assistir filmes
<option value = "Esporte"> Praticar esportes
</select>


Observações gerais:
<textarea name = "Observação" rows = "5" cols = "60"></textarea>

<hr>
<p>
<input type = "submit" value = "Enviar os dados">
<input type = "reset" value = "Limpar os dados">
</p>
<hr/>
</form>

O formulário começa com a tag <form> e tudo dentro dele fará parte do formulário.
O atributo action é vazio, pois os dados serão enviados para a própria página.
O método utilizado é POST, ou seja, os dados serão enviados para o corpo da requisição "http", normalmente é usado para envio de dados sensíveis ou em caso de operações que possam alterar o estado do servidor.
Se for o método for GET, então os dados do fomulário são anexados à URL do site. Normalmente é usado em sites que usam busca e consulta onde os dados não são importantes

Dentro do formulário, há campos para o nome completo, e-mail, sexo, bens (como casa, automóvel, micro), hobby preferido (com opções como ler livros, ouvir música, etc.) e observações gerais.

Os campos de texto para nome e e-mail são criados com a tag <input> do tipo text.
Os botões de RADIO para selecionar o sexo são criados com a tag <input> do tipo RADIO.
Os checkboxes para selecionar bens são criados com a tag <input> do tipo checkbox.
O campo de seleção para o hobby preferido é criado com a tag <select>, contendo várias opções dentro de tags <option>.
O campo de observações gerais é criado com a tag <textarea>.
Finalmente, há botões de envio (<input type="submit">) e de limpeza (<input type="reset “>) dentro de um parágrafo alinhado ao centro.


Inicializando para a web retorna a esse formulário a direita.
